# **Scribd-Viewer-JS**

Scribd-Viewer-JS is an extension of the Scribd-Viewer that allows users to easily access this tool using any devices as long as it have a supported web browser, and the good thing is that it does not requires the user to download anything to use this program as everything is web-based!
